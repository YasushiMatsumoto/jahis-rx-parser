## 実装上の書式ルール

### 日付

- 西暦: `YYYYMMDD`
- 和暦: `GYYMMDD`
- 年号 `G` は `M/T/S/H/R`

### 郵便番号

- `NNNNNNN`
- `NNN-NNNN`

### 電話番号

- 数字とハイフンで最大13文字

### 体重

- 最大3桁整数 + 小数点 + 最大3桁小数
- 例: `63.7`, `0.85`, `123.456`

### 用量

- 最大6桁整数 + 小数点 + 最大5桁小数
- 例: `100`, `1.5`, `123456.78901`
